## .startswith()
- Checks weather a string begins with a specified prefix.

    🦋 
    text = "hello world"

    print(text.startswith("hello"))   ✅ Output: True
    print(text.startswith("world"))   ✅ Output: False
    print(text.startswith("h"))       ✅ Output: True
